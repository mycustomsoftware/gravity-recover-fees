Gravity Forms Fee Plugin

Plugin information

The "Gravity Forms Fee" plugin enables customers to cover additional fees associated with their purchases when using the Gravity Forms platform. It seamlessly integrates with the Gravity Forms plugin, providing customers with the option to include any applicable fees during the checkout process. This functionality streamlines transactions and enhances user experience by offering transparency and flexibility in payment options.

Instructions
  1. Activate plugin
  2. Add field Recover Fee in Gravity Forms
  3. Add fix price and parent for recover fee
  4. Label with variable %RECOVERFEE% that will be replaced with calculated fee instead Example:
    • Help cover our transaction fees %RECOVERFEE% so that 100% of your donation goes to those in need.
  5. If you need setup default values for inputs simple got to Forms -> Settings -> Fee Add-On and fulfill them.
  6. If you what to change label in admin you can click on field in form and got to Settings
